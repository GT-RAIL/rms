      Create the HTML and Javascript needed to create a multi-stream MJPEG canvas with the given information.

      create_multi_mjpeg_canvas_by_envid(integer $envid, integer $width, integer $height, integer $default, string | null $cb) : string

      This will create an HTML5 canvas element to render on and thus should be placed in the BODY section of the HTML. Note that this returns the HTML, you will need to echo it in the appropriate spot.

      Parameters

      $envid

      integer

      The environment to get the list of MJPEG streams from

      $width

      integer

      The width of the canvas/stream

      $height

      integer

      The height of the canvas/stream

      $default

      integer

      The index of the default stream (0 by default)

      $cb

      stringnull

      The name of a JavaScript function that will be called once the widget is created or null if no callback is being used (default is null)

      Returns

      stringThe HTML to create the widget and the slider
